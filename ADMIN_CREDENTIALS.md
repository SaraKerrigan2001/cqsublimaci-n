### 🚀 Cómo Usar

#### Para Registrar un Nuevo Admin:
1. Ve a la página principal
2. Haz clic en "Usuario" → "Registrarse"
3. Completa el formulario con el diseño oscuro
4. Selecciona "Administrador" en el tipo de usuario
5. Haz clic en "Registrarse"

#### Para Acceder como Admin:
1. Ve a la página principal
2. Haz clic en "Admin" → "Acceso Admin"
3. Usa las credenciales:
   - Email: `admin@cqsublimacion.com`
   - Contraseña: `admin123`
4. Haz clic en "Acceder como Admin" (botón naranja)
5. Serás redirigido automáticamente a `/admin`

#### Para Ejecutar el Proyecto:
```bash
npm run dev
```
Luego ve a: `http://localhost:3000`
